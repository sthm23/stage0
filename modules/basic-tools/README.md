# Stage#0. Basics of working with Chrome Dev Tools and VS Code
## In this module, students need
Get acquainted with:

   - features of the browser
   - Chrome Dev Tools (Styles and Element Inspector)
   - VSCode editor
        - environment setup
        - extensions
        - console
   - Basic terminal commands

4 hours
